
## Git 개념
버전 관리 시스템
  
  
## Git의 기능
1. Backup  
만약의 사태에 대비할 수 있다.
2. Recovery  
이전 상태로 돌아갈 수 있다.
3. Collaboration  
다른 사람과 협업이 가능하다.
